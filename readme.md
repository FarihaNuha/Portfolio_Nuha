# Fariha Tasnim Nuha - A Personal Portfolio Website

A modern and responsive portfolio website showcasing the skills, projects, and achievements of Fariha Tasnim Nuha, a student of Educational Technology & Engineering.

## Live Website
**Visit:** [https://portfolio-nuha.vercel.app/]

##  Project Description
This portfolio website features a professional dark theme with gradient accents, smooth animations, and interactive elements. It presents educational background, technical skills, project showcases, and achievements in an engaging visual format. The website is fully responsive and works on all devices.

##  Technologies Used

### **Frontend:**
_**HTML5:**_ Semantic markup for structure

_**CSS3:**_ Custom styling with modern features (Flexbox, Grid, Animations)

_**JavaScript:**_ Interactive features and form validation

### Libraries & Tools:
_**Font Awesome 6.4.0:**_ Icon library for UI elements

_**Google Fonts (Inter):**_ Modern typography

_**Responsive Design:**_ Media queries for various screen sizes

## Sections Included
1. **Home** - Introduction with typing animation
2. **About Me** - Personal information and interests
3. **Education** - Academic timeline
4. **Skills** - Technical and creative skills with progress bars
5. **Projects** - Showcase of completed projects
6. **Achievements** - Certificates and awards
7. **Contact** - Contact form and social media links

## Main Features
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices
- **Interactive Navigation:** Smooth scrolling with active section highlighting
- **Project Showcase:** Gallery-style display of projects with hover effects
- **Skills Visualization:** Animated progress bars for skill proficiency
- **Contact Form:** Interactive form with validation
- **Mobile Menu:** Hamburger menu for mobile devices
- **Typography Animation:** Name typing effect on homepage
- **Certificate Gallery:** Display of achievements and certifications
- **Social Media Integration:** Direct links to professional profiles
- **Downloadable CV**
